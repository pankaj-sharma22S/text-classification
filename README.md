# SMS Spam Detection

A machine learning project that classifies SMS messages as Spam or Ham.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- TF-IDF
- Multinomial Naive Bayes

## Accuracy
96.4%

## Project Structure
├── spam_detection.ipynb
├── spam.csv
├── spam_model.pkl
